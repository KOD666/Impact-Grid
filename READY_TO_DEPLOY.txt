  - pnpm (or npm/yarn)

Commands:
  pnpm install          # Install dependencies (already done)
  pnpm run dev          # Start dev server → http://localhost:3000
  pnpm run build        # Production build
  pnpm run start        # Start production server
  pnpm run type-check   # Type check
  pnpm run lint         # Lint code

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

Issue: Port 3000 already in use
  Solution: pnpm run dev -- -p 3001

Issue: Module not found errors
  Solution: pnpm install && pnpm run dev

Issue: Leaflet map not rendering
  Solution: Check browser console for CORS issues, verify Leaflet CSS loaded

Issue: Deploy not working
  Solution: Verify /api/deploy endpoint returns 200, check pendingChanges state

For detailed debugging:
  - Check Console (F12) for client errors
  - Check Terminal for server errors
  - Review Network tab for failed requests
  - Check Redux/AppContext state in React DevTools
